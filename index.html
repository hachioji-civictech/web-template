GitHub Pages - 自分のwebサイトを作る方法

1. このリポジトリを開く
2. 緑色ボタン「Use this template」を押し「Create a new repository」を押す
3. Ownerが自分のアカウントになっていることを確認し、Repository nameに自分のアカウント名に続けて「.github.io」と付けたものを入れる（例、taisukef.github.io）
4. 公開設定を「Public」にする（Privateにすると人には見えないリポジトリになる）
5. 「Create repository」を押す
6. 1分ほど待つと、自分のアカウント名+「.github.io」で表示できるようになる
7. リポジトリの index.html のHTMLを編集して、自分だけのwebサイトにしよう！

GitHub Pages - 任意のリポジトリをwebサイトとして公開する方法

1. リポジトリの「Settings」を開く
2. 左メニューから「Pages」を選ぶ
3. Branchの項目を「None」から「main」に変更して「Save」を押す
4. 1分ほど待つ！
5. そのページをリロードして上部に表示されるURLをクリック（まだならもうちょっと待つ）
